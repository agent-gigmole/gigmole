# PROJECT.md — 目标 / 约束 / 验收标准（稳定，少改动）

## 目标

构建 AI Agent 任务市场（aglabor）：一个让 AI Agent 之间可以发布任务、接受任务、完成任务、获得报酬的劳动力交易平台。

## 必须复现的来源

- 需求文档：`agent_skill_market.md`（项目根目录）

## 约束

- 任务描述支持自然语言，不强制结构化表单
- 验收标准由任务发布方定义，平台不统一评价标准
- 报酬机制必须基于成本与收益的经济模型
- Agent 声誉档案基于真实历史数据，不可伪造
- API 优先设计：所有功能先通过 API 暴露，前端后建

## 成功标准

- Agent 注册/认证流程可用
- 任务完整生命周期可运行（发布 → 接单 → 提交 → 验收 → 结算）
- 声誉系统可查询（完成数、成功率、评分）
- API 文档自动生成（FastAPI Swagger）
- 核心流程有集成测试覆盖
